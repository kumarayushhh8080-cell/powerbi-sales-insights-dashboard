 Overview

This project is an interactive Sales Dashboard built using Power BI. It provides insights into sales performance, customer behavior, delivery efficiency, and order trends for an e-commerce dataset.

⸻

🎯 Objectives
	•	Analyze overall sales performance
	•	Track delivery delays and efficiency
	•	Understand customer ratings trend
	•	Identify preferred payment methods
	•	Monitor order status (Delivered, Returned, Cancelled)

⸻

📈 Key KPIs
	•	💰 Total Sales
	•	📦 Total Orders
	•	⭐ Average Rating
	•	🚚 Average Delivery Days
	•	💵 Average Sales per Order

⸻

📊 Dashboard Features
	•	Sales Trend Analysis (Monthly)
	•	Rating Trend Visualization
	•	Delivery Delay by State
	•	Payment Method Share (Donut Chart)
	•	Order Status Distribution
	•	Interactive Filters & Slicers

⸻

🛠️ Tools & Technologies
	•	Power BI
	•	DAX (Data Analysis Expressions)
	•	Data Modeling
	•	Data Visualization

⸻

🧠 Key Learnings
	•	Created calculated columns and measures using DAX
	•	Built interactive and dynamic dashboards
	•	Handled real-world data issues like duplicates and missing values
	•	Improved data storytelling and visualization skills
